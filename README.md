# SMART_PLANTS

Descriere:

     Proiectul “SMART_PLANTS” este un sistem autonom și sustenabil care se folosește de panouri solare pentru a reîncărca bateriile ce îl pun în mișcare, al cărui scop este de a servi în agricultura. Acesta consta într-un sistem de irigare și un sistem de detectare și stingere al focului, astfel eliminand nevoia constanta de ingrijire de către fermier și eliminand grijile unui incendiu care ar putea avea consecințe catastrofale.
